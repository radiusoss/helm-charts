
1. Create a port-forward to the Zeppelin pod and growse with your favorite browser http://localhost:8080:

   kubectl port-forward -n {{ .Release.Namespace }} $(kubectl get pod -n {{ .Release.Namespace }} --selector=app={{ template "zeppelin.name" . }} -o jsonpath='{.items...metadata.name}') 8080:8080

2. If you want to run a Spark job or debug Zeppelin, launch a shell in the Zeppelin pod:

   kubectl exec -n {{ .Release.Namespace }} -it $(kubectl get pods -n {{ .Release.Namespace }} --selector=app={{ template "zeppelin.name" . }} -o jsonpath="{.items[0].metadata.name}") -c zeppelin -- bash
